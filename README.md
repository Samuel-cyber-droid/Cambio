3. Puntos de Inflexión (Red Flags)

Durante la ejecución de este Roadmap, se debe prestar atención inmediata si ocurren los siguientes eventos:

Baja participación en el Sandbox (Fase 2): Indica una resistencia pasiva o falta de tiempo. Se debe ajustar la carga laboral para permitir la capacitación.

Prolongación de la Operación Dual (Fase 3): Si los usuarios se niegan a soltar el sistema viejo después de 2 semanas, el "Hard Stop" de la Fase 4 debe ejecutarse por mandato directivo.
